# 176. Second Highest Salary

**Data:** 21/12/2025  
**Dificuldade:** Medium  
**Tópicos:** SQL, CTE, DENSE_RANK, Window Function, ORDER BY  
**Link:** https://leetcode.com/problems/second-highest-salary/description/?envType=study-plan-v2&envId=top-sql-50

## Descrição
Escreva uma solução para encontrar o **segundo maior salário** na tabela `Employee`.  
Caso não exista um segundo maior salário, o resultado deve ser **NULL**.

## Solução (SQL Server)

```sql
WITH ranked_salaries AS (
    SELECT DISTINCT
        id,
        salary,
        DENSE_RANK() OVER (ORDER BY salary DESC) AS salary_rank
    FROM Employee
)
SELECT
    MAX(salary) AS SecondHighestSalary
FROM ranked_salaries
WHERE salary_rank = 2;
