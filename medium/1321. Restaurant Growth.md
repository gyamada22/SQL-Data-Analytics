# 1321. Restaurant Growth

**Data:** 30/12/2025
**Dificuldade:** Medium  
**Tópicos:** SQL, Window Functions, Aggregation  
**Link:** Restaurant Growth - LeetCode

## Descrição
Write a solution to calculate the **7-day moving sum and average** of daily customer spending.

Only return records where a **full 7-day window** is available.

---

## Solução (SQL Server)

```sql
WITH daily_sales AS (
    SELECT
        visited_on,
        SUM(amount) AS daily_amount
    FROM customer
    GROUP BY visited_on
),
rolling_metrics AS (
    SELECT
        visited_on,
        SUM(daily_amount) OVER (
            ORDER BY visited_on
            ROWS BETWEEN 6 PRECEDING AND CURRENT ROW
        ) AS amount,
        ROUND(
            AVG(CAST(daily_amount AS FLOAT)) OVER (
                ORDER BY visited_on
                ROWS BETWEEN 6 PRECEDING AND CURRENT ROW
            ),
            2
        ) AS average_amount,
        ROW_NUMBER() OVER (ORDER BY visited_on) AS row_num
    FROM daily_sales
)
SELECT
    visited_on,
    amount,
    average_amount
FROM rolling_metrics
WHERE row_num >= 7
ORDER BY visited_on;
