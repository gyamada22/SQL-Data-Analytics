# 608. Tree Node

**Data:** 19/01  
**Dificuldade:** Medium  
**Tópicos:** SQL, CASE, Subquery  
**Link:** https://leetcode.com/problems/tree-node/description/?envType=problem-list-v2&envId=vszzv79m  

## Descrição
O objetivo é classificar cada nó de uma árvore em três tipos possíveis:

- **Root**: quando o nó não possui pai (`p_id IS NULL`);
- **Inner**: quando o nó possui ao menos um filho (ou seja, seu `id` aparece como `p_id` de outro nó);
- **Leaf**: quando o nó não é raiz e não possui filhos.

Cada linha da tabela representa um nó da árvore, e devemos identificar corretamente o seu tipo com base nessas regras.

## Solução (SQL Server)
```sql
SELECT 
    id,
    CASE
        WHEN p_id IS NULL THEN 'Root'
        WHEN id IN (SELECT DISTINCT p_id FROM tree) THEN 'Inner'
        ELSE 'Leaf'
    END AS type
FROM tree;
