# 178. Rank Scores

**Data:** 22/01/2026    
**Dificuldade:** Medium  
**Tópicos:** SQL, Window Function, DENSE_RANK, ORDER BY  
**Link:** https://leetcode.com/problems/rank-scores/description/?envType=problem-list-v2&envId=vszzv79m  

## Descrição
O objetivo é atribuir um ranking para cada score de forma que:

- Scores maiores recebam ranks menores (melhor posição).
- Scores iguais compartilhem o mesmo rank.
- Não existam “buracos” na numeração do ranking após empates.

Esse comportamento é exatamente o fornecido pela função de janela `DENSE_RANK()`.

---

## Solução (SQL Server)
```sql
SELECT  
    score,
    DENSE_RANK() OVER (ORDER BY score DESC) AS rank
FROM scores
ORDER BY score DESC;
