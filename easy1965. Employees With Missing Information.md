# 1965. Employees With Missing Information

**Data:** 11/01/2026  
**Dificuldade:** Easy  
**Tópicos:** SQL, LEFT JOIN, UNION ALL, Data Consistency  
**Link:** https://leetcode.com/problems/employees-with-missing-information/description/

## Descrição
Escreva uma solução para encontrar os funcionários que possuem **informações faltando** entre as tabelas `Employees` e `Salaries`.

Um funcionário é considerado com informação faltante quando:
- Existe em `Salaries`, mas não existe em `Employees`, ou
- Existe em `Employees`, mas não existe em `Salaries`.

Ou seja, qualquer `employee_id` que não possua correspondência nas duas tabelas.

---

## Solução (SQL Server)

```sql
SELECT employee_id
FROM (
    -- Funcionários que existem em Salaries, mas não em Employees
    SELECT sal.employee_id
    FROM Salaries sal
    LEFT JOIN Employees emp
        ON sal.employee_id = emp.employee_id
    WHERE emp.name IS NULL

    UNION ALL

    -- Funcionários que existem em Employees, mas não em Salaries
    SELECT emp.employee_id
    FROM Employees emp
    LEFT JOIN Salaries sal
        ON emp.employee_id = sal.employee_id
    WHERE sal.salary IS NULL
) AS missing_info
ORDER BY employee_id;
