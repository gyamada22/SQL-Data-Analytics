# 185. Department Top Three Salaries

**Data:** 04/01/2026  
**Dificuldade:** Hard  
**Tópicos:** SQL, Window Functions, DENSE_RANK, JOIN  
**Link:** [Department Top Three Salaries - LeetCode](https://leetcode.com/problems/department-top-three-salaries/description/?source=submission-ac)

## Descrição
Escreva uma solução para listar os **três maiores salários de cada departamento**.  
Caso existam empates de salário, **todos os funcionários empatados** devem ser incluídos no resultado.

## Solução (SQL Server)

```sql
WITH ranked_salaries AS (
    SELECT
        e.*,
        DENSE_RANK() OVER (
            PARTITION BY e.departmentId
            ORDER BY e.salary DESC
        ) AS salary_rank
    FROM employee e
)
SELECT
    d.name AS Department,
    rs.name AS Employee,
    rs.salary AS Salary
FROM ranked_salaries rs
JOIN department d
    ON rs.departmentId = d.id
WHERE rs.salary_rank <= 3;
