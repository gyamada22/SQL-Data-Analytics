# 465. Find Products with Valid Serial Numbers

**Data:** 15/01/2026  
**Dificuldade:** Easy  
**Tópicos:** SQL, REGEXP, String Matching  
**Link:** [Find Products with Valid Serial Numbers - LeetCode](https://leetcode.com/problems/find-products-with-valid-serial-numbers/description/)

## Descrição
Encontre todos os produtos cuja coluna `description` contenha um **número de série válido** no formato:
SNXXXX-YYYY

Onde:
- `SN` deve estar em maiúsculo  
- `XXXX` e `YYYY` são números com exatamente 4 dígitos  
- Deve respeitar limites de palavra (não pode estar “grudado” em outras letras ou números)

## Solução (SQL)

```sql
SELECT *
FROM products
WHERE description REGEXP '(?-i)\\bSN[0-9]{4}-[0-9]{4}\\b'
ORDER BY product_id;
