# 1729. Find Followers Count

**Data:** 14/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, GROUP BY, COUNT  
**Link:** https://leetcode.com/problems/find-followers-count/description/?envType=study-plan-v2&envId=top-sql-50

## Descrição
A tabela `Followers` contém as colunas:
- `user_id`
- `follower_id`

Cada linha indica que o usuário `follower_id` segue o usuário `user_id`.

Escreva uma solução SQL para **retornar o número de seguidores que cada usuário possui**.

O resultado deve conter:
- `user_id`
- `followers_count`

O resultado deve ser ordenado por `user_id` em ordem crescente.

## Solução (SQL Server)

```sql
select user_id,
        count(follower_id) as followers_count
from Followers
group by user_id
order by user_id asc
