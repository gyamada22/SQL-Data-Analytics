# 610. Triangle Judgement

**Data:** 16/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, CASE  
**Link:** https://leetcode.com/problems/triangle-judgement/

## Descrição
Cada linha da tabela contém três segmentos de reta. Escreva uma solução para informar se esses três segmentos podem formar um triângulo.

## Solução (SQL Server)

```sql
SELECT  x,
        y,
        z,
        CASE
            WHEN (x > 0 AND y > 0 AND z > 0)
             AND (x < y + z)
             AND (y < x + z)
             AND (z < x + y) THEN 'Yes'
            ELSE 'No'
        END AS triangle
FROM Triangle;
