# 1141. User Activity for the Past 30 Days I

**Data:** 14/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, GROUP BY, COUNT, WHERE  
**Link:** https://leetcode.com/problems/user-activity-for-the-past-30-days-i/?envType=study-plan-v2&envId=top-sql-50

## Descrição
Escreva uma solução SQL para encontrar o número de **usuários ativos distintos por dia** nos últimos **30 dias**, considerando como data final **2019-07-27**.

A tabela `Activity` contém:
- `user_id`
- `session_id`
- `activity_date`
- `activity_type`

Um usuário é considerado ativo em um dia se realizou pelo menos uma atividade nesse dia.

O resultado deve conter:
- `day` (data da atividade)
- `active_users` (quantidade de usuários distintos ativos no dia)

## Solução (SQL Server / MySQL)

```sql
SELECT
    activity_date AS day,
    COUNT(DISTINCT user_id) AS active_users
FROM Activity
WHERE activity_date BETWEEN '2019-06-28' AND '2019-07-27'
GROUP BY activity_date;
