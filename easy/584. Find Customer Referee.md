# 584. Find Customer Referee

**Data:** 11/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, WHERE, NULL Handling, Comparison Operators  
**Link:** [Find Customer Referee - LeetCode](https://leetcode.com/problems/find-customer-referee/)

## Descrição
Encontre o nome dos clientes cujo referee_id é nulo ou diferente de 2.

## Solução (SQL Server)

```sql
select name
from customer
where referee_id is null or referee_id <> 2
