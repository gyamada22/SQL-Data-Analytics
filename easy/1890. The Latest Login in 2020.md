# 1890. The Latest Login in 2020

**Data:** 10/01/2026  
**Dificuldade:** Easy  
**Tópicos:** SQL, Window Functions, CTE, DENSE_RANK  
**Link:** https://leetcode.com/problems/the-latest-login-in-2020/description/

## Descrição
Escreva uma solução para encontrar, para cada usuário, o **último login realizado no ano de 2020**.

Se um usuário tiver feito vários logins em 2020, devemos retornar apenas o mais recente.

---

## Solução (SQL Server)

```sql
WITH latest_logins AS (
    SELECT 
        lg.user_id,
        lg.time_stamp,
        DENSE_RANK() OVER (
            PARTITION BY lg.user_id 
            ORDER BY lg.time_stamp DESC
        ) AS login_rank
    FROM Logins lg
    WHERE YEAR(lg.time_stamp) = 2020
)
SELECT 
    user_id,
    time_stamp AS last_stamp
FROM latest_logins
WHERE login_rank = 1;
