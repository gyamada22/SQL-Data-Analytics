# 1683. Invalid Tweets

**Data:** 12/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, String Functions, WHERE  
**Link:** [Invalid Tweets - LeetCode](https://leetcode.com/problems/invalid-tweets/)

## Descrição
Escreva uma solução para encontrar os IDs dos tweets inválidos. Um tweet é inválido se o número de caracteres usados no conteúdo do tweet for estritamente maior que 15. Retorne a tabela de resultados em qualquer ordem.

## Solução (SQL Server)

```sql
select tweet_id
from tweets 
where len(content) > 15
