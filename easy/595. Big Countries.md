# 595. Big Countries

**Data:** 11/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, WHERE, OR Operator  
**Link:** [Big Countries - LeetCode](https://leetcode.com/problems/big-countries/)

## Descrição
Um país é considerado grande se tiver área de pelo menos 3 milhões de km² OU população de pelo menos 25 milhões. Escreva uma solução para encontrar os países grandes.

## Solução (SQL Server)

```sql
select name, population, area
from World
where area >= 3000000 or population >= 25000000
