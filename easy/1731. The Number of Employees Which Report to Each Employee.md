# 1731. The Number of Employees Which Report to Each Employee

**Data:** 14/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, JOIN, GROUP BY  
**Link:** https://leetcode.com/problems/the-number-of-employees-which-report-to-each-employee/description/?envType=study-plan-v2&envId=top-sql-50

## Solução
```sql
SELECT
    ee.employee_id,
    ee.name,
    COUNT(e.reports_to) AS reports_count,
    ROUND(AVG(e.age * 1.0), 0) AS average_age
FROM Employees e
JOIN Employees ee
    ON e.reports_to = ee.employee_id
GROUP BY
    ee.employee_id,
    ee.name
ORDER BY
    ee.employee_id;
