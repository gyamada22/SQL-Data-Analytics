# 1824. Find Users With Valid E‑Mails

**Data:** 16/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, LIKE, SUBSTRING  
**Link:** https://leetcode.com/problems/find-users-with-valid-e-mails/description/?envType=study-plan-v2&envId=top-sql-50

## Descrição
Escreva uma solução para retornar `user_id`, `name` e `mail` de usuários cujo email começa com letra, termina com `@leetcode.com` e contém apenas caracteres válidos antes do domínio.

## Solução (SQL Server)

```sql
with qwe as(
    select  user_id,
            name,
            mail
    from users
    where left(mail,1) like '[a-zA-Z]'
      and right(mail,13) COLLATE Latin1_General_CS_AS = '@leetcode.com'
      and substring(mail,1, (len(mail) - 13))  not like '%[^a-zA-Z0-9._-]%'
)

select  user_id,
        name,
        mail
from qwe
