# 1581. Customer Who Visited but Did Not Make Any Transactions

**Data:** 12/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, LEFT JOIN, GROUP BY, COUNT, NULL Handling  
**Link:** [Customer Who Visited but Did Not Make Any Transactions - LeetCode](https://leetcode.com/problems/customer-who-visited-but-did-not-make-any-transactions/description/)

## Descrição
Escreva uma solução para encontrar o ID de cada cliente e o número de visitas em que o cliente não fez nenhuma transação. Retorne a tabela de resultados ordenada por `count_no_trans` em ordem decrescente.

## Solução (SQL Server)

```sql
select v.customer_id, count(v.visit_id) as count_no_trans
from Visits v left join Transactions t on v.visit_id = t.visit_id
where t.transaction_id is null
group by v.customer_id
