# 1407. Top Travellers

**Data:** 08/01/2026  
**Dificuldade:** Easy  
**Tópicos:** SQL, JOIN, GROUP BY, ISNULL, ORDER BY  
**Link:** [Top Travellers - LeetCode](https://leetcode.com/problems/top-travellers/description/?envType=problem-list-v2&envId=vbrhdq5g)

## Descrição
Calcule a **distância total percorrida por cada usuário** e liste-os do maior para o menor.  
Inclua também usuários que **não fizeram nenhuma viagem**, mostrando distância zero.

## Solução (SQL Server)

```sql
SELECT 
    u.name,
    ISNULL(SUM(r.distance), 0) AS travelled_distance
FROM Users u
LEFT JOIN Rides r 
    ON u.id = r.user_id
GROUP BY u.id, u.name
ORDER BY travelled_distance DESC, name;
