# 586. Customer Placing the Largest Number of Orders

**Data:** 24/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, GROUP BY, COUNT, TOP 1  
**Link:** [Customer Placing the Largest Number of Orders - LeetCode](https://leetcode.com/problems/customer-placing-the-largest-number-of-orders/?source=submission-ac)

## Descrição
Encontre o **cliente que fez o maior número de pedidos** na tabela `Orders`.  
- Se houver empate, apenas um cliente será retornado.  

## Solução (SQL Server)

```sql
with qwe as (
    select  customer_number,
            count(order_number) cnt
    from orders
    group by customer_number
)


select top 1 customer_number
from qwe
order by cnt desc;
