# 182. Duplicate Emails

**Data:** 23/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, GROUP BY, HAVING  
**Link:** [Duplicate Emails - LeetCode](https://leetcode.com/problems/duplicate-emails/description/?source=submission-ac)

## Descrição
Escreva uma solução para reportar todos os emails que aparecem mais de uma vez na tabela `Person`.

## Solução (SQL Server)

```sql
WITH qwe AS (
    SELECT email, COUNT(email) AS cnt_email
    FROM Person
    GROUP BY email
    HAVING COUNT(email) > 1
)
SELECT email
FROM qwe;
