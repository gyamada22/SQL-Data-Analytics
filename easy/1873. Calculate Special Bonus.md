# 1873. Calculate Special Bonus

**Data:** 09/01/2026  
**Dificuldade:** Easy  
**Tópicos:** SQL, CASE, Condições Lógicas  
**Link:** https://leetcode.com/problems/calculate-special-bonus/

## Descrição
Escreva uma solução para calcular o **bônus especial de cada funcionário** com base nas seguintes regras:

- O `employee_id` deve ser **ímpar**
- O nome do funcionário **não pode começar com a letra 'M'**
- Se ambas as condições forem atendidas, o bônus será igual ao `salary`
- Caso contrário, o bônus será `0`

---

## Solução (SQL Server)

```sql
SELECT  
    emp.employee_id,
    CASE
        WHEN emp.employee_id % 2 <> 0 
             AND LEFT(emp.name, 1) <> 'M' 
        THEN emp.salary
        ELSE 0
    END AS bonus
FROM Employees emp
ORDER BY emp.employee_id;
