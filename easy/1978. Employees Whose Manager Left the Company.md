# 1978. Employees Whose Manager Left the Company

**Data:** 16/12/2025  
**Dificuldade:** Easy  
**Tópicos:** SQL, Subquery  
**Link:** https://leetcode.com/problems/employees-whose-manager-left-the-company/

## Descrição
Escreva uma solução para encontrar os funcionários cujo salário é menor que 30000 e cujo gerente não trabalha mais na empresa.

## Solução (SQL Server)

```sql
SELECT employee_id
FROM Employees
WHERE salary < 30000
  AND manager_id NOT IN (
        SELECT employee_id
        FROM Employees
  )
ORDER BY employee_id ASC;
